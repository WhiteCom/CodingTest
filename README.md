# CodingTest
- 코딩 테스트 대비를 위한 3인 공동 주 1회 백준 문제풀이 Repository 입니다.

## 코딩테스트에 잘 나오는 항목
- 구현(수학)
- 스택(Stack)
- 큐(Queue)
- 연결리스트(LinkedList)
- 해쉬(Hash)
- 이분탐색 (Binary Search)
- 완전탐색(Brute Froce)
- 그리디(Greedy)
- 동적계획법(DP : Dynamic Programming)
- 문자열처리
- 최단경로(Shortest Path)
- 정렬(Sorting)
- 그래프(DFS/BFS)
- 재귀(Recursion)
- 백트래킹(BackTracking)

## 사용 언어
- C++
- Python
- JAVA(JDK >= 8)
